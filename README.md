# Control-del-ordenador-con-gestos-medici-n-de-mano-Computer-Vision-
Este proyecto implementa un sistema de interacción humano-computador basado en visión artificial que permite controlar el cursor del sistema operativo mediante gestos de la mano capturados por una webcam, además de estimar medidas físicas de la mano en centímetros utilizando calibración geométrica con un marcador ArUco.
